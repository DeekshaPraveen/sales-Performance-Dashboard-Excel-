# 📊 Sales Performance Dashboard (Excel)

## 📌 Project Overview

Developed an interactive **Sales Performance Dashboard** using **Microsoft Excel** to analyze key business metrics such as revenue, profit, regional performance, and product trends. This project converts raw sales data into meaningful visual insights, enabling smarter and faster business decisions.

---

## 🎯 Objectives

* Evaluate overall sales and profit performance
* Identify top-performing regions and products
* Analyze sales trends over time
* Enable dynamic and interactive data exploration

---

## 🛠 Tools & Technologies

* **Microsoft Excel**
* **Pivot Tables**
* **Pivot Charts**
* **Slicers (Interactive Filters)**
* **Data Cleaning & Preparation**

---

## ⚙️ Key Features

* 📌 **KPI Metrics Dashboard** – Total Sales, Total Profit, and Growth Indicators
* 📈 **Time-Based Trend Analysis** – Monthly / Quarterly sales performance tracking
* 🌍 **Region-Wise Insights** – Compare sales and profit across locations
* 🛍 **Product Performance Analysis** – Best-selling and underperforming products
* 🎛 **Interactive Dashboard** – Easy filtering with slicers for better analysis
